# AI Video Course

把 YouTube 上较长、偏技术的 AI Engineering 视频，重构成适合进阶学习的结构化课程。

## 为谁服务

默认用户有基础开发项目经验，能够使用 AI 进行 vibe coding，但希望进一步理解技术原理、架构设计、工程取舍和实践边界。

这不是英语学习产品。默认使用中文讲解；英文讲解与中文字幕是按需切换的呈现方式。

## 核心学习流程

```text
YouTube 链接
→ 课程导读：值不值得学、重点在哪里
→ 知识地图：按主题拆分，而非机械切片
→ 模块学习：观点、通俗解释、技术机制、案例、实践
→ 时间戳问答：回答可回到原视频核对
→ 课程重讲：中文默认，也可切换英文讲解
```
